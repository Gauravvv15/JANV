# JANV v0.15 (Frozen)

JANV is a personal Python-based assistant project.

## Current Status
This version (v0.15) is **frozen** and stable.

### Implemented Features
- One-time core initialization (`janv_core.json`)
- Command execution system (apps / URLs)
- Persistent response system (`responses.json`)
- Basic conversational structure
- Data safety and initialization guards

## Development Note
Further development is **paused intentionally** to allow skill upgrades.

JANV will resume development after sufficient learning.
